# [irvingmg.com](https://irvingmg.com/)

Personal blog for notes on learning, projects, and technical references.

## Tech Stack

- [Hugo](https://gohugo.io/) - Static site generator
- [Paper](https://github.com/nanxiaobei/hugo-paper) - Theme

## Local Development

```bash
# Clone with submodules
git clone --recurse-submodules https://github.com/IrvingMg/IrvingMg.github.io.git

# Run dev server
hugo server
```

## Deployment

Automatically deployed to GitHub Pages on push to `main`.
